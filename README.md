Firefly Focus

Firefly Focus is a small educational game prototype developed using Phaser 3, TypeScript, and Vite.

The game is designed as a visual attention and object tracking exercise inspired by vision-training activities. Players must remember a highlighted firefly and continue tracking it while multiple fireflies move around the screen.

Features
Built with Phaser 3 + TypeScript
Animated forest environment
Firefly sprite animations
Countdown before tracking begins
Multiple difficulty levels
Score system
Increasing challenge by adding more moving objects
Focus and visual attention training mechanics
Gameplay
A firefly is highlighted.
The player memorizes the target.
After a countdown, all fireflies start moving.
The player must identify the original firefly.
Difficulty increases as levels progress.
Purpose

This prototype was created to explore interactive exercises that may help with:

Visual attention
Object tracking skills
Concentration and focus exercises

This game is intended as an educational prototype and does not make medical claims regarding eyesight improvement.
